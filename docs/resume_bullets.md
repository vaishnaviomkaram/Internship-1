# Resume Bullets — AskMyBook

## Draft Resume Bullets

- Built a Retrieval-Augmented Generation document Q&A system using Python, LangChain, Chroma, HuggingFace embeddings, and Gemini, enabling citation-backed answers from uploaded PDFs.

- Implemented hybrid BM25 + dense retrieval and strict prompt guardrails, improving answer relevance and reducing hallucination on document-specific questions.

- Developed multi-document comparison, quiz generation, and LLM-as-Judge evaluation modes; deployed the app on Streamlit Cloud and added pytest-based unit tests for ingestion and chunking.
